# Innovatex-Dashboard
Business Performance Dashboard using Power BI
🎯 Objective
To analyze business performance metrics including sales, revenue trends, and key operational KPIs.
🛠 Tools Used
Power BI
DAX
Power Query
📊 Key Features
KPI cards for revenue, profit, and growth
Sales trend analysis over time
Regional and category performance breakdown
Interactive filters for dynamic analysis
🔍 Key Insights
Identified top-performing regions contributing to overall revenue
Highlighted declining trends in specific product categories
Enabled stakeholders to track KPIs in real-time
